================================================================================
 Screenshot Tool — Release Notes
================================================================================

--------------------------------------------------------------------------------
 v1.0.1 — 2026-04-24
--------------------------------------------------------------------------------
  - Added app icon (ICO + PNG) to the exe, editor window title bar,
    desktop shortcut, Start Menu entry, and Inno Setup installer wizard

--------------------------------------------------------------------------------
 v1.1.0 — 2026-04-24
--------------------------------------------------------------------------------
  - System tray icon — app stays running in the background after launch;
    right-click tray for New Snip or Exit
  - Global hotkey now uses Win32 RegisterHotKey API (more reliable in compiled builds)
  - Hotkey (Ctrl+Shift+S) works even when app is minimized to tray
  - Editor window minimizes before capture so it doesn't appear in the screenshot
  - Tool tooltips — hover over any tool button to see its name
  - Default window size set to 1100x700 so all tools are visible on open
  - Minimum window size enforced at 800x550

--------------------------------------------------------------------------------
 v1.0.0 — 2026-04-24
--------------------------------------------------------------------------------
Initial release.

Features:
  - Click-and-drag region capture with fullscreen darkened overlay
  - Live selection size readout while dragging
  - Auto-saves every capture to ~/Pictures/Screenshots/ with a timestamp
  - Global hotkey: Ctrl+Shift+S triggers a new snip from anywhere
  - Annotation editor with:
      - Pen (freehand drawing)
      - Marker (semi-transparent highlight with opacity slider)
      - Eraser — erase to white OR erase to original screenshot
      - Line, Arrow, Rectangle, Ellipse
      - Text tool
  - 10 preset colors + custom color picker
  - 6 brush sizes (2–24 px)
  - Undo / Redo (Ctrl+Z / Ctrl+Y, up to 50 steps)
  - Copy annotated image to clipboard (Ctrl+C)
  - Save As dialog (Ctrl+S)
  - New Snip button — capture again without closing the editor
  - GitHub Releases auto-updater — blue banner appears in the editor when a
    newer version is available; one-click install and restart
  - Inno Setup installer — no admin rights required (installs to LocalAppData)
  - build.bat — one-command PyInstaller + installer + zip packaging
